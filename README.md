# Bar Amasa Cocktail Trainer

A simple training website built for the Bar Amasa team to help memorise cocktail builds.

## Updating the Menu

To update cocktail information:

1. Edit the relevant drink data in the project files
2. Save and commit your changes
3. Push to GitHub
4. GitHub Pages will publish the updated version

## Project Goal

The goal of this project is to make cocktail training more efficient, practical, and easy to revisit, without relying only on printed notes or verbal run-throughs.
